## API Reference

#### Get all schedule

```http
  GET /api/schedule
```

#### Get all list member

```http
  GET /api/member/
```

#### Get detail member

```http
  GET /api/member/:idmember
```

| Parameter  | Type     | Description                       |
| :--------- | :------- | :-------------------------------- |
| `idmember` | `number` | **Required**. Id of item to fetch |
